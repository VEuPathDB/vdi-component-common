fun DatasetID(raw: String): DatasetID

Attempts to construct a new DatasetID instance from the given string value.

If the given string value could not be a valid DatasetID value, an IllegalArgumentException will be thrown.

For a string value to appear valid, it must be a 32 digit string consisting of only hex characters.

For example:

  • 78609580dba2787a5cf677d6f334a707

  • A5FA4B2B38398ADD73D5929F3DCFDEA8

Return

A DatasetID instance parsed from the given raw string.

Parameters

raw

Raw string value that should be parsed as a DatasetID instance.

Throws

IllegalArgumentException

If the given raw string is not 32 characters in length, or contains a non-hex character.

fun DatasetID(): DatasetID

Generates a new dataset, random dataset ID.

Generated IDs are not guaranteed to be 100% unique, however the chance of collision with another dataset ID is 1 in 2.71 quintillion, and thus may be safely considered unique for most applications.

Return

A new random dataset ID.
