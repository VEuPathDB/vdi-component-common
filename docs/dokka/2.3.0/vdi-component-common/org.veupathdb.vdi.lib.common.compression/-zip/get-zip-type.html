fun Path.getZipType(): ZipType

Returns the type of Zip archive the target path points to.

If the target points to a file that is not a valid zip archive, the value ZipType.Invalid will be returned.

Receiver

Target path to test.

Return

Zip type for the target path, or ZipType.Invalid if the target path is not a valid Zip archive.
