fun Environment.optMap(key: String): Map<String, String>?

Retrieves an optional target Map environment variable or null if the target variable is blank or absent.

Map environment variables are defined as key/value pairs in a string value in the following format:

key1:value,key2:value,key3:value
Content copied to clipboard

Note: Keys may not contain a : character, however values may.

Return

A map of the raw string values present in the environment variable.

Parameters

key

Key of the target environment variable to look up.

Throws

IllegalStateException

If the environment variable is not in the correct map format.
